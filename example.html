Marketo Form Block Example

This page demonstrates how the Marketo Form Block plugin can be used to embed Marketo forms in WordPress using the Gutenberg editor.

Basic Usage

To add a Marketo form to your WordPress content:

  1. Edit a post or page using the Gutenberg editor
  2. Click the "+" button to add a new block
  3. Search for "Marketo Form" and select the block
  4. In the block settings sidebar, enter your Marketo Form ID
  5. Configure additional options as needed

Custom CSS Example

Here's an example of custom CSS you can use to style the form:

/* Change form width */
.mktoForm {
  width: 100% !important;
  max-width: 500px;
}

/* Style form labels */
.mktoForm .mktoLabel {
  color: #333;
  font-weight: 600;
}

/* Style form fields */
.mktoForm .mktoField {
  border: 1px solid #ddd;
  border-radius: 4px;
  padding: 10px;
}

/* Style submit button */
.mktoForm .mktoButton {
  background-color: #0073aa;
  color: white;
  border: none;
  padding: 10px 20px;
  border-radius: 4px;
}
